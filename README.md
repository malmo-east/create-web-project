# create-web-project

🥃 Create web site with no build configuration.

# 💿 Installation
```
$ npm init
$ npm install
```

# 📖 Usage
## Commands
- npm start

- gulp sass
- gulp browser-sync
- gulp scripts
- gulp css-libs
- gulp watch
- gulp clean
- gulp img
- gulp build
- gulp clear
